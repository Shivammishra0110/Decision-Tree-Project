# Decision-Tree-Project
## Overview
Built a Decision Tree-based classification model with an end-to-end pipeline including data preprocessing, feature selection, model training, and performance evaluation using Python and Scikit-learn.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Model Training (Decision Tree)
5. Model Evaluation

## Objective
To build a machine learning model capable of making accurate predictions using a Decision Tree algorithm.

## Project File
- Decision_Tree_Project.ipynb

##  How to Run
1. Clone the repository
2. Open the notebook in Jupyter Notebook or Google Colab
3. Run all cells

##  Results
Achieved 97% accuracy on test data
